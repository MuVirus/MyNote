# 二叉树
## 定义
### 简单二叉树结构（裸指针）
``` cpp
struct TreeNode {
    int val;
    TreeNode *left;
    TreeNode *right;
    TreeNode(int x) : val(x), left(NULL), right(NULL) {}
    ~TreeNode() {
        if (left != nullptr) {
            delete left;
            left = nullptr;
        }
        if (right != nullptr) {
            delete right;
            right = nullptr;
        }
    }    
};
```


## 遍历
### 前中后序遍历
**前序遍历**：根-左-右。先获取根节点值，再访问根的左子树，最后访问根的右子树。
**中序遍历**：左-根-右。先访问根的左子树，再获取根节点值，最后访问根的右子树。
**后序遍历**：左-右-根。先访问根的左子树，再访问根的右子树，最后获取根节点值。
### 递归法
#### 前/中/后序遍历
##### 通用模板（前序遍历为模板）
``` cpp
class Solution {
public:
    void traversal(TreeNode* node, vector<int>& vec) {
        if(node == nullptr) return;
        vec.push_back(node->val);//根据根的位置移动这一行即可。
        traversal(node->left, vec);
        traversal(node->right, vec);
    }
    vector<int> preorderTraversal(TreeNode* root) {
        vector<int> ans;
        traversal(root, ans);
        return ans;
    }
};
```
注意
- 递归写法最好单独放到一个函数中去写，因为可以按照自己需求定义函数的返回值/参数，避免一些内存浪费。
##### lambda写法（C++11）（中序遍历为例）
``` cpp
    vector<int> inorderTraversal(TreeNode* root) {
        vector<int> vec;
        function<void(TreeNode* node)> traversal = [&](TreeNode* node){
            if(node == nullptr) return;
            traversal(node->left);
            vec.push_back(node->val);
            traversal(node->right);
        };
        traversal(root);
        return vec;
    }
```

### 迭代法
#### 前/中序遍历
##### 通用模板
``` cpp
class Solution {
public:
    vector<int> preorderTraversal(TreeNode* root) {
        vector<int> vec;
        stack<TreeNode*> st;
        while(root || !st.empty()) {
            if(root) {
                st.push(root);
                // vec.push_back(root->val); // 前序遍历（先根）
                root = root->left; // 当root有效时，优先遍历左节点
            } else {
                root = st.top();
                st.pop();
                // vec.push_back(root); // 中序遍历（先左）
                root = root->right; // 当root左子树遍历完时，再遍历右节点
            }
        }
        return vec;
    }
};
```
#### 后序遍历
后续遍历的顺序是：**左-右-根**，所以说我们只需要按照**根-右-左**的顺序进行遍历得到数组，最后翻转一下即可。
整体模板还是按照`先序遍历`来写，因为原本顺序为`根左右`，不过翻转之后为`右左根`，不符合要求，所以只需要修改一下为先遍历右节点即可。
``` cpp
class Solution {
public:
    vector<int> postorderTraversal(TreeNode* root) {
        vector<int> vec;
        stack<TreeNode*> st;
        while(root || !st.empty()) {
            if(root) {
                st.push(root);
                vec.push_back(root->val);
                root = root->right; // 当root有效时，优先遍历右节点
            } else {
                root = st.top();
                st.pop();
                root = root->left; // 当root右子树遍历完时，再遍历右节点
            }
        }
        reverse(vec.begin(), vec.end()); // 最后翻转
        return vec;
    }
};
```


## 题型
### 对称二叉树

``` cpp
class Solution {
public:
    bool compare(TreeNode* left, TreeNode* right) {
        if(left == nullptr && right == nullptr) return true;
        else if(left == nullptr && right)   return false;
        else if(left && right == nullptr)   return false;
        if(left->val != right->val) return false;
        auto outside = compare(left->left, right->right);
        auto inside = compare(left->right, right->left);
        return (outside && inside);
    }
    bool isSymmetric(TreeNode* root) {
        return compare(root->left, root->right);
    }
};
```
