# 二叉树
## 定义
### 简单二叉树结构（裸指针）
``` cpp
struct TreeNode {
    int val;
    TreeNode *left;
    TreeNode *right;
    TreeNode(int x) : val(x), left(NULL), right(NULL) {}
    ~TreeNode() {
        if (left != nullptr) {
            delete left;
            left = nullptr;
        }
        if (right != nullptr) {
            delete right;
            right = nullptr;
        }
    }    
};
```


## 遍历
### 前中后序遍历
**前序遍历**：根-左-右。先获取根节点值，再访问根的左子树，最后访问根的右子树。
**中序遍历**：左-根-右。先访问根的左子树，再获取根节点值，最后访问根的右子树。
**后序遍历**：左-右-根。先访问根的左子树，再访问根的右子树，最后获取根节点值。
### 递归法
#### 前/中/后序遍历
##### 通用模板（前序遍历为模板）
``` cpp
class Solution {
public:
    void traversal(TreeNode* node, vector<int>& vec) {
        if(node == nullptr) return;
        vec.push_back(node->val);//根据根的位置移动这一行即可。
        traversal(node->left, vec);
        traversal(node->right, vec);
    }
    vector<int> preorderTraversal(TreeNode* root) {
        vector<int> ans;
        traversal(root, ans);
        return ans;
    }
};
```
注意
- 递归写法最好单独放到一个函数中去写，因为可以按照自己需求定义函数的返回值/参数，避免一些内存浪费。
##### lambda写法（C++11）（中序遍历为例）
``` cpp
    vector<int> inorderTraversal(TreeNode* root) {
        vector<int> vec;
        function<void(TreeNode* node)> traversal = [&](TreeNode* node){
            if(node == nullptr) return;
            traversal(node->left);
            vec.push_back(node->val);
            traversal(node->right);
        };
        traversal(root);
        return vec;
    }
```
