# 二分查找
## 定义

### 朴素二分法
> 重要：
> 就算遇到左右端点问题，也可以靠朴素法解决。
#### 代码模板1（l<=r）
用来找顺序数组中的target所在的序号
``` cpp
int search(vector<int>& nums, int target) {
    int n = nums.size();
    int l = 0, r = n - 1;
    while(l <= r) {
        int mid =  l + ((r - l) >> 1);
        if(nums[mid] < target) {
            l = mid + 1;
        } else if(nums[mid] > target) {
            r = mid - 1;
        } else {
            return mid;
        }
    }
    return -1;
}
```
- `l + ((r - l) >> 1)`是为了避免溢出的现象，等价于`(l + r) >> 1`
- 不能直接使用`(r >> 1) + (l >> 1)`，可能会损失
- 注意：`while(l <= r)`只是其中一种写法，用习惯最好就别变了。
#### 代码模板（l<r）
``` cpp
int search(vector<int>& nums, int target) {
    int n = nums.size();
    // 处理空数组的特殊情况
    if (n == 0) return -1;
    
    int l = 0, r = n - 1;
    while(l < r) {
        int mid =  l + ((r - l) >> 1);
        if(nums[mid] < target) {
            // 目标在右侧，左边界移到 mid+1（和原逻辑一致）
            l = mid + 1;
        } else {
            // 关键修改：nums[mid] >= target 时，右边界移到 mid（而非 mid-1）
            // 保证不会错过目标值，同时避免死循环
            r = mid;
        }
    }
    // 循环结束时 l == r，检查该位置是否是目标值
    return nums[l] == target ? l : -1;
}
```
#### 解决左右边界问题（朴素法）
>题1：[34. 在排序数组中查找元素的第一个和最后一个位置](https://leetcode.cn/problems/find-first-and-last-position-of-element-in-sorted-array/description/?envType=study-plan-v2)
>题2：[35. 搜索插入位置](https://leetcode.cn/problems/search-insert-position/description/?envType=study-plan-v2)
##### 思路
朴素二分法的目的是找到那个target所在的序号，所以按照这个思路走下去。
- 如果能找到对应的target
	- nums为严格有序数组，则target对应的序号`唯一`
	- nums非严格有序数组，则target对应的序号`可能`不为唯一。
		- 由于当`nums[mid] == target`时，mid就是nums中元素为target的其中一个
			- 解决办法：以mid为中间，创建midL、midR(或直接复用mid)，初始值为mid，分别向左向右检查，直接就可以检测到左右边界了（见题一）。
- 如果不能找到对应的target
	- 解决办法：就得用到`while(l <= r)`的性质了，根据题目最后返回`l`还是`r`
	- 如果要找正好大于target（向右第一个大于），大概率返回l
	- 如果要找正好小于target（向左第一个小于），大概率返回r
##### 题一代码
``` cpp
vector<int> searchRange(vector<int>& nums, int target) {
    int n = nums.size();
    int l = 0, r = n - 1;
    while(l <= r) {
        int mid = l + ((r - l) >> 1);
        if(nums[mid] < target) {
            l = mid + 1;
        } else if(nums[mid] > target) {
            r = mid - 1;
        } else {
            // 向左右扫描
            int mid_left = mid;
            while(mid_left >= 0 && nums[mid_left] == target) {
                --mid_left;
            }
            while(mid < n && nums[mid] == target) {
                ++mid;
            }
            return {mid_left + 1, mid - 1};
        }
    }
    return {-1, -1};
}
```

##### 题二代码
``` cpp
int searchInsert(vector<int>& nums, int target) {
    int n = nums.size();
    int l = 0, r = n - 1;
    if(target <= nums[0])   return 0;
    while(l <= r) {
        int mid = (l + r) >> 1;
        if(nums[mid] < target) {
            ++l;
        } else if(nums[mid] > target) {
            --r;
        } else {
            while(mid >= 0 && nums[mid] == target) {
                --mid;
            }
            return ++mid;
        }
    }
    return l;
}
```

``` cpp
int takeAttendance(vector<int>& records) {
    int n = records.size();
    int l = 0, r = n - 1;
    if(records[0])  return 0;   // 先判断头部
    while(l <= r) {
        int mid = l + ((r - l) >> 1);
        if(records[mid] == mid) {
            // 如果值与序号相等，则左边没问题
            l = mid + 1;
        } else {
            // 如果不相等，就当作找到了，找到第一个即可
            while(mid > 0 && records[mid] != mid) {
                --mid;
            }
            return ++mid;
        }
    }
    return l;
}
```
