# 二分查找
## 定义

### 朴素二分模板
> 重要：
> 就算遇到左右端点问题，也可以靠朴素法解决。

``` cpp
    int search(vector<int>& nums, int target) {
        int n = nums.size();
        int l = 0, r = n - 1;
        while(l <= r) {
            int mid =  l + ((r - l) >> 1);
            if(nums[mid] < target) {
                l = mid + 1;
            } else if(nums[mid] > target) {
                r = mid - 1;
            } else {
                return mid;
            }
        }
        return -1;
    }
```
