## 安装

通过github去下载就行了，直接git clone或者下载zip压缩包。
然后运行.\bootstrap-vcpkg.bat即可。

## 环境变量
参考这两行命令，这两行是用于临时系统变量，永久设置差不多。
```
$env:VCPKG_ROOT = "C:\path\to\vcpkg"
$env:PATH = "$env:VCPKG_ROOT;$env:PATH"
```
