# 主谓
- S + V
- 主语 + 不及物动词
```
The moon rose.
月亮升起。
```
# 主谓宾
- S + V + O
- 主语 + 谓语动词 + 宾语
```
Jack bought a car.
杰克买了一辆车。
```
# 主谓宾补
- S + V + O + C
- 主语 + 谓语动词 + 宾语 + 宾语补语
```
They painted the house blue.
他们把房间涂成蓝色。
```
# 主谓间宾直宾
- S + V + IO + DO
- 主语 + 谓语动词 + 间接宾语 + 直接宾语
```
Jack lent Mark money.
杰克借马克钱。
```
# 主谓补
- S + V + C
- 主语 + （连）系动词 + 表语
```
She is happy.
她很开心。
```
