| 阶段           | 核心内容           | 为什么必须先学？                               |
| ------------ | -------------- | -------------------------------------- |
| 1. **微观经济学** | 供需、弹性、价格机制     | 商业世界的 "原子结构"，理解市场如何运作、价格如何形成，所有商业决策的基础 |
| 2. **宏观经济学** | GDP、通胀、货币政策    | 理解经济 "大环境" 如何影响你的行业和生意，避免 "只见树木不见森林"   |
| 3. **行为经济学** | 认知偏误、消费心理      | 打破 "人是理性的" 幻想，理解真实消费行为（为什么限时促销让人冲动下单）  |
| 4. **财务基础**  | 利润表、现金流、成本分析   | 避免 "账上赚钱，实际没钱" 的悲剧，掌握企业生存的 "血液循环系统"    |
| 5. **市场营销**  | 用户洞察、定价策略、渠道选择 | 解决 "有好产品却卖不出去" 的难题，将理论转化为实战            |
| 6. **商业模式**  | 盈利逻辑、价值创造      | 整合前 5 步，设计可持续赚钱的系统（如平台模式、订阅制）          |
| 7. **基础法律**  | 合同、股权、劳动法      | 避开创业路上的法律陷阱，保护你的 "劳动果实"                |
| 8. **行业知识**  | 按兴趣选择领域        | 落地到具体场景，将理论转化为解决实际问题的能力                |
