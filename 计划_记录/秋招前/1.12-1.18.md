# 嵌入式软件

## MCU

### 开源库
#### 1、MultiButton

> 多按键管理库，支持短按、长按、多次按，用状态机管理状态。

- [x] 试验了BasicExample，简单使用uwTick来零阻塞延时5ms检查一次。
- [x] 研究代码
#### 2、SEGGER RTT

> 一种高速通信方法，几乎不会导致MCU延迟，一般用于实时传输。

- [x] 下载并安装了Jlink驱动，并且找到了SEGGER的C库，并移植到了STM32的代码中，调试方面用PyOCD跟STM32进行RTT通信。

#### 3、MultiTimer

> 一种软件定时器管理库，可以拓展出很多定时器，对定时精度要求不高可以用。

- [x] 添加了多个任务，进行定时执行（用到uwTick进行阻塞式延时1ms，任务用到外设又led灯和马达）
- [x] 研究代码
