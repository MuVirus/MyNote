特性：
- **非阻塞：** 启动定时器后，调用线程/协程立刻返回，不因等待而阻塞
- **异步回调：** 超时后(设置的定时值)自动执行用户指定的回调函数
- **可重复使用：** 同一个定时器对象可以多次启动，重新绑定新的任务
- **线程安全：** 可在多线程环境下安全调用
- **资源安全：** 遵循RAII原则

核心成员

| 成员        | 类型    | 作用                  |
| --------- | ----- | ------------------- |
| worker_   | 线程/协程 | 工作线程/协程，负责时间等待与回调执行 |
| active_   | bool  | 当前是否有定时任务           |
| stopped_  | bool  | 是否正在销毁              |
| expiry_   |       | 到期时间点               |
| callback_ |       | 用户回调函数              |
